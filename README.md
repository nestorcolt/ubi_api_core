# DevOps Engineer Test App

### How to run:

```
docker-compose up --build
```

### Local development:


HealthAPI
```
http://localhost:8080/health
```


OpenAPI
```
http://localhost:8080/openapi
```


Swagger UI
```
http://localhost:3000/?url=http://localhost:8080/openapi
```
